CoderDojo Software Installation Cheatsheet (v1.0)

CoderDojo Info

  • Main club website: http://coderdojodc.com; general info at http://coderdojo.com
  • Meetup group at http://www.meetup.com/CoderDojoDC/
  • Mailing list at https://groups.google.com/forum/#!forum/coderdojodc

Scratch

The current version of Scratch, Scratch 2.0, is a web app. A beta version of 2.0 that allows offline work can be downloaded here:

  • http://scratch.mit.edu/scratch2download/

The previous version, Scratch 1.4, works well without wifi access and can still be downloaded from:

  • http://scratch.mit.edu/scratch_1.4/

Twine

Twine is a framework for creating interactive fiction games (in the spirit of "Choose Your Own Adventure" books, but with many powerful features for adding interactive elements to your stories.

  • http://twinery.org

Python

We've had a variety of programming activities at the Dojo in the past, but the 'main' technologies that we've repeatedly come back to have been Python, Twine, and Scratch.

  • http://www.python.org/getit/

You can write Python programs in any text editor, but very soon after beginning Python, you'll probably find it helpful to work with a more full-featured editor than just Notepad or TextEdit. There are many free ones to choose from, but one good option is Komodo Edit:

  • http://www.activestate.com/komodo-edit

While CoderDojo doesn't have a formal curriculum, in the past we've used the book Invent with Python by Al Sweigart to work through some basic Python concepts. It is available both in hardcopy and for free online:

  • http://inventwithpython.com

Minecraft

Many of our participants are interested in Minecraft, and we tried to introduce programming activities into the Minecraft space through MCEdit and ScriptCraft. MCEdit is an open-source world editor for Minecraft. ScriptCraft is a framework for modding Minecraft using Javascript.

  • http://scriptcraftjs.org
  • http://www.mcedit.net
  • To use either of these, you'll also need a version of Minecraft installed: https://minecraft.net.

Arduino

  • Arduino boards are microcontrollers designed for hobbyists and inventors.
  • More info at: http://arduino.cc/en/Main/HomePage
